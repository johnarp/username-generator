# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.2.0] - 2026-01-06

### Added

- Large `.txt` word lists from [stuff](https://github.com/hugsy/stuff) GitHub repository by [hugsy](https://github.com/hugsy)
- [CREDITS](CREDITS.md) file
- Changing and saving themes with JavaScript and Material Icons
- Icon for clearing LocalStorage
- "Copy failed" prompt for mobile devices

### Changed

- Word generation now uses the word lists
- Separate CSS variable for button text
- [README](README.md) instructions for opening in a browser

### Fixed

- v0.1.0 Changelog date format

### Removed

- Hardcoded word arrays

## [0.1.0] - 2026-01-05

### Added

- Site and styling
- Buttons for generating and copying
- PyScript for username generation
- Message when load site and copy username
- Footer with GitHub link
- Placeholder adjectives and nouns
